JsonSerializerOptions.PropertyNamingPolicy 属性的说明

JsonSerializerOptions.PropertyNamingPolicy 属性的官方说明：
获取或设置一个值，该值指定用于将对象的属性名称转换为其他格式（例如 camel 大小写）的策略；若为 null，则保持属性名称不变。
